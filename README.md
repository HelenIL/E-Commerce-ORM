## Installation

- Requirements: Node.js and MySQL

- Clone repository: `https://github.com/HelenIL/E-Commerce-ORM.git`

- Run `npm install` from root directory

- Run `npm start`

## Languages/Dependencies/Resources

* MySQL
* Sequelize
* Node.js

* Dotenv
